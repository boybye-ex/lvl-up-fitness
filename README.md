# LVL UP FITNESS

A gamified workout tracker web app built with React + Vite.

## Features

- **Gamified Progress**: XP, levels, and achievements
- **Cloud Sync**: Firebase authentication and real-time data sync
- **AI Coach**: Powered by Google Gemini for personalized advice
- **Skill Tree**: Unlock new workouts as you progress
- **Personal Records**: Track and celebrate PRs

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Create a `.env` file with your API keys:
   ```
   VITE_GEMINI_API_KEY=your_gemini_api_key
   ```

3. Start the dev server:
   ```bash
   npm run dev
   ```

## Build for Production

```bash
npm run build
```

## Tech Stack

- React 19
- Vite
- Tailwind CSS v4
- Firebase (Auth & Firestore)
- Google Gemini AI
- Framer Motion
